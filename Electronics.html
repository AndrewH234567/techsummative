Electronics

During the electronics unit, I built circuits on tinkercad and in real life. I also coded these circuits to do different things such as morse code or follow the pentiometer. Each one was called a "circuit task"

Assorted circuits

This circuit task was just an introduction to get to know all the circuit type. I also built a parrelel circuit in real life.

Dimmable LED

This circuit task was to build a circuit in which you could dim and brighten an LED by adjusting the petentiometer. I built this in real life.

RGB LED

This circuit task is where I built a circuit with three petentiometers with 1 RGB LED, and each petentiometer adjusts a different colour. I built this circuit in real life.

Morse Code

I sadly don't have an image for this one, but this task was just coding an led to blink something in morse code. I used the word "skibidi"

Chaser lights

This one was by far the hardest, where I had to code and build a circuit in which when you turned the petentiometer, the chaser lights get faster/slower.
